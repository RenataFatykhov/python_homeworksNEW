1. Метод `capitilize`:
   - Описание: Метод не корректно обрабатывает строки, начинающиеся с символов, отличных от букв.
   - Шаги для воспроизведения:
     1. Вызвать capitilize("123abc").
     2. Ожидаемый результат: "123abc".
     3. Фактический результат: "123abc".
   - Рекомендуемое решение: Добавить проверку на первый символ.

2. Метод `trim`:
   - Описание: Не удаляет пробелы в конце строки.
   - Шаги для воспроизведения:
     1. Вызвать trim("  skypro  ").
     2. Ожидаемый результат: "skypro".
     3. Фактический результат: "skypro  ".
   - Рекомендуемое решение: Использовать strip.

3. Метод `to_list`:
   - Описание: Не обрабатывает пустые строки с разделителем корректно.
   - Шаги для воспроизведения:
     1. Вызвать to_list("a,,b", ",").
     2. Ожидаемый результат: ["a", "", "b"].
     3. Фактический результат: ["a", "b"].
   - Рекомендуемое решение: Учитывать пустые строки между разделителями.

4. Метод `contains`:
   - Описание: Не различает регистр символов.
   - Шаги для воспроизведения:
     1. Вызвать contains("SkyPro", "p").
     2. Ожидаемый результат: False.
     3. Фактический результат: True.
   - Рекомендуемое решение: Добавить опцию для учета регистра.
